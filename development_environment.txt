# 개발 환경
**리눅스 + Windows
리눅스(우분투) - Server
Windows 10 - 임시 Client

**자바
OpenJDK 8

**파이썬
Python 3.6.8
KoNLPy - 한국어 정보처리
scikit-learn - 머신러닝
ChatterBot - 키워드 matching




# 자바 설치 (리눅스)
**APT 업데이트
sudo apt-get update

**자바 설치
sudo apt-get install openjdk-8-jdk
sudo apt-get install openjdk-8-jre (jre만 원하는 경우)

**여러 종류의 자바가 설치되어 있을시 Default Java 설정
sudo update-alternatives --config java

**자바 버전 확인
java -version
javac -version




# 파이썬 설치 (리눅스)
**APT 업데이트
sudo apt-get update

**Pyton 및 pip 설치
sudo apt-get install python3
sudo apt-get install python3-pip

**Pyton 버전 확인
python3 --version




# 필요한 파이썬 패키지 설치
**KoNLPy 설치
pip3 install jpype1
pip3 install konlpy

**scikit-learn 설치
pip3 install scikit-learn

**ChatterBot 을 위한 패키지들
pip3 install spacy<br>
pip3 install pytz<br>
pip3 install pint<br>
pip3 install mathparse<br>
pip3 install sqlalchemy<br>
pip3 install nltk<br>
pip3 install date_parser<br><br>

**Path
